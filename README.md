# BaseChange

BaseChange is a utility for converting numbers between arbitrary bases (2-36). It supports both integer and fractional parts, and handles negative numbers.

## Overview

The converter module implements base conversion functionality that allows users to transform numeric values from one base to another. It supports bases from 2 (binary) to 36 (alphanumeric), where digits 10-35 are represented by letters A-Z.

## Features

- Conversion between any two bases (2-36)
- Support for fractional parts (decimal points)
- Support for negative numbers
- Input validation to ensure numbers conform to the source base
- Size validation to prevent overflow

## Usage

Run the converter with:

```bash
python converter.py
